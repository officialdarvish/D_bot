# UI v27 - Referral, Test Account, Setup and Happ Guide Fix

## Bot setup
- After initial setup, the admin/owner immediately receives the rules confirmation page.
- After accepting rules, the main menu opens normally.

## Test Account
- Test-account volume and duration remain editable from Admin Web > Test Account and Admin Bot settings.
- Existing single-user test-account usage removal by Telegram ID is preserved.

## Referral system
- Referral links now attach both new users and existing users who have no referrer yet.
- Referrers receive the referred user's details when a new subset is registered.
- Purchase commission uses the configured percentage globally.
- Commission is applied after successful wallet/card/crypto purchases.
- The referrer must have at least one paid purchase before receiving commission.
- Referrers receive a purchase/commission message plus a wallet balance message.

## My Configs
- Refresh/update keeps the same service-detail message instead of sending a new result message.
- Old inline messages show a short /start notification instead of running stale actions.

## Delivery guide
- Public, test-account, and reseller delivery messages now strongly recommend only Happ.
- The guide includes Happ import steps and 12-hour subscription refresh instructions.
